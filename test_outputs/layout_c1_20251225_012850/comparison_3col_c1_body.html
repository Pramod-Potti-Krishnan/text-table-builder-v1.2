Basic Service Tier

Target User: Ideal for individuals or small teams just getting started with our platform.

Core Features: Access to all standard tools and essential project management boards.

Support Level: Standard email support with a guaranteed 48-hour response time.

Data Analytics: Basic dashboard providing a high-level overview of key metrics.

Integrations: Limited to 3 standard third-party application integrations.

Security: Standard two-factor authentication and at-rest data encryption.

Onboarding: Self-service setup with access to our online knowledge base and guides.

Professional Service Tier

Target User: Designed for growing businesses and established teams needing more power.

Core Features: All Basic features plus advanced automation workflows and user permissions.

Support Level: Priority email and live chat support with a 24-hour response time.

Data Analytics: Customizable reports and advanced analytics with data export options.

Integrations: Unlimited standard integrations and full access to our developer API.

Security: Includes role-based access controls and a comprehensive audit log history.

Onboarding: A guided one-on-one session with a dedicated customer success specialist.

Enterprise Service Tier

Target User: Built for large organizations requiring full scalability and compliance.

Core Features: All Pro features plus custom branding, data residency, and sandboxing.

Support Level: 24/7 dedicated support manager and a guaranteed 99.9% uptime SLA.

Data Analytics: Predictive analytics, AI-driven insights, and business intelligence tool sync.

Integrations: Includes custom integration development and premium API support services.

Security: Single Sign-On (SSO) support and advanced, industry-specific compliance.

Onboarding: A comprehensive, tailored implementation plan and on-site team training.
